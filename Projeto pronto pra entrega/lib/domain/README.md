# domain

Camada de regras de negócio.

No Muscleway, regras de negócio incluem:

- validar login;
- cadastrar usuário sem duplicar e-mail;
- calcular total do carrinho;
- calcular desconto no Pix;
- definir quando há frete grátis;
- decidir quais produtos a IA pode recomendar.

Em uma versão futura, essas regras podem ser separadas em casos de uso.
