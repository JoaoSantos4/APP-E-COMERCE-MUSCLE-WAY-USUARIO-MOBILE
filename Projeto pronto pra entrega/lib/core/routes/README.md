# core/routes

Camada para centralizar rotas do app.

Rotas atuais no Muscleway:

- `/` - login;
- `/cadastro` - cadastro;
- `/home` - tela principal;
- `/produto` - detalhes do produto;
- `/carrinho` - carrinho;
- `/chat-ia` - consultor de IA.

Hoje as rotas estão em `app_muscley/lib/src/app.muscley.dart`.
