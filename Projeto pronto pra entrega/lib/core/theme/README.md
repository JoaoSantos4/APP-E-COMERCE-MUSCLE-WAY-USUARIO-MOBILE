# core/theme

Camada para tema visual do aplicativo.

No app atual, o tema está em:

```text
app_muscley/lib/src/theme/app_theme.dart
```

Ele define:

- fundo escuro;
- cor principal azul bebê;
- cor de superfície;
- estilo de AppBar;
- estilo de botões;
- estilo de campos de formulário;
- estilo de SnackBar.
