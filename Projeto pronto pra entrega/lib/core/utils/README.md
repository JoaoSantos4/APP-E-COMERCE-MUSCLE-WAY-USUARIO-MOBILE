# core/utils

Camada para funções utilitárias.

No Muscleway, exemplos de utilidades são:

- formatar preço em reais;
- calcular preço com desconto no Pix;
- normalizar texto para busca;
- validar valores comuns.

No código atual, a classe `PrecoUtils` fica em:

```text
app_muscley/lib/src/utils/preco_utils.dart
```
