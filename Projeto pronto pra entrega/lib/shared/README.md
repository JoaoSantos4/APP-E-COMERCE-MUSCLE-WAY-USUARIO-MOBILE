# shared

Camada para componentes e recursos compartilhados.

Exemplos:

- widgets reutilizáveis;
- estilos comuns;
- helpers;
- mensagens padrão;
- componentes que podem ser usados em várias telas.

No projeto atual, `BotaoWidget`, `BrandLogoWidget`, `CampoFormularioWidget` e
`ProdutoCardWidget` são exemplos de componentes compartilhados.
