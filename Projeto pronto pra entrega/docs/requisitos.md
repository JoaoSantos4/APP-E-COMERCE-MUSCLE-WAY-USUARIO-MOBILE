# Requisitos Funcionais

## RF01 - Cadastro de usuário

O sistema deve permitir que o usuário crie uma conta informando nome, e-mail,
senha e confirmação de senha.

## RF02 - Login de usuário

O sistema deve permitir login com e-mail e senha cadastrados.

## RF03 - Validação de formulário

O sistema deve validar campos obrigatórios, formato de e-mail e tamanho mínimo
da senha.

## RF04 - Catálogo de produtos

O sistema deve exibir produtos de suplementos cadastrados localmente.

## RF05 - Busca de produtos

O sistema deve permitir buscar produtos por nome, categoria ou descrição.

## RF06 - Filtro por categoria

O sistema deve permitir filtrar produtos por categorias como proteínas,
aminoácidos, pré-treinos, kits e acessórios.

## RF07 - Detalhes do produto

O sistema deve exibir uma tela com detalhes, preço, descrição e benefícios do
produto.

## RF08 - Carrinho de compras

O sistema deve permitir adicionar, remover, aumentar e diminuir a quantidade de
produtos no carrinho.

## RF09 - Resumo de checkout

O sistema deve exibir subtotal, total de itens, entrega e botão de finalizar
compra.

## RF10 - Consultor de IA

O sistema deve permitir que o usuário envie perguntas para receber sugestões de
suplementos.

## RF11 - Integração com OpenAI

O sistema deve enviar perguntas e catálogo para o backend, que consulta a API da
OpenAI.

## RF12 - Fallback local de IA

Se a IA real não responder, o sistema deve gerar uma sugestão local baseada em
palavras-chave.

## RF13 - Logout

O sistema deve permitir sair da conta e limpar a sessão atual.
