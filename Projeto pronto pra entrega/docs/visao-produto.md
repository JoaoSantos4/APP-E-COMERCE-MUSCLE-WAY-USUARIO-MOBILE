# Visão do Produto

## Nome

Muscleway.

## Descrição

O Muscleway é um aplicativo de e-commerce para venda de suplementos, vitaminas,
kits e acessórios esportivos. O objetivo é oferecer uma experiência parecida com
uma loja real, com catálogo organizado, carrinho de compras e um consultor de IA
que ajuda o cliente a escolher produtos conforme seu objetivo.

## Problema

Pessoas que compram suplementos nem sempre sabem qual produto combina melhor com
seu objetivo, como ganhar massa, emagrecer, ter mais energia ou melhorar a
recuperação muscular.

## Solução

O app reúne produtos em categorias e adiciona um consultor de IA. O usuário pode
perguntar o que comprar, e a IA sugere opções baseadas no catálogo disponível.

## Público-alvo

- alunos de academia;
- praticantes de musculação;
- pessoas que buscam suplementação esportiva;
- clientes que querem montar kits de produtos;
- usuários que precisam de ajuda para escolher suplementos.

## Diferenciais

- identidade visual própria;
- carrinho funcional;
- catálogo com categorias;
- busca por produto;
- IA generativa via API;
- fallback local para quando a API não estiver disponível;
- backend separado para proteger a chave da OpenAI.

## Objetivos do projeto

- Demonstrar uso de Flutter em um app completo.
- Demonstrar persistência local com SQLite.
- Demonstrar integração com IA generativa.
- Demonstrar organização de código e documentação de projeto.
- Simular um produto próximo de uma aplicação real.
